Модуль “Ripple Effect”

Категорії: Документація «Чертоги Фрілансера» v3.0.0

Модуль додає до будь-якого об’єкту ефект хвилі при кліку

Підключення

[HTML] До будь-якого об’єкту додайте дата атрибут data-ripple

<button type="submit" data-ripple class="button">Надіслати</button>

[SCSS] В файлі src/scss/base.scss розкоментуйте рядок @import “base/ripple”; Файл src/scss/base/ripple.scss містить необхідні стилі для ефекту. Ці стилі можна змінювати під себе, наприклад час анімації:

.ripple {
	   /* ... код ... */
		animation: button-circle 1.5s ease forwards;
	}

[JS] у файлі src/js/app.js розкоментуйте рядок flsFunctions.rippleEffect();
